# 🏨 BookMyStay

**BookMyStay** is an **Airbnb-inspired hotel booking platform** built using the **MERN stack (MongoDB, Express, React, Node.js)**.

🚧 **Project Status:** Work in Progress

This project is inspired by Airbnb and aims to provide a platform where users can explore hotels, view details, and book stays easily.

---

## ✨ Features (Planned / In Progress)

- 🔐 User Authentication (Signup / Login)
- 🏨 Browse available hotels
- 📄 Hotel details page
- 📅 Room booking system
- ⭐ Reviews and ratings
- ❤️ Save hotels to favorites
- 📊 User booking dashboard

---

## 🛠️ Tech Stack

- ⚛️ React (Frontend)
- 🎨 Tailwind CSS
- 🟢 Node.js
- 🚂 Express.js
- 🍃 MongoDB
- 🔑 JWT Authentication

---

## 📂 Project Structure

    BookMyStay

        ├── frontend
        └── backend

- **frontend** → React application (UI)
- **backend** → Node.js + Express API

## 🎯 Purpose

This project is being built to:

- Understand real-world application architecture
